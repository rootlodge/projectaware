# neversleep.ai UI

This is the Next.js + Tailwind CSS frontend for the neversleep.ai agent.

## Features
- Voice-to-text input (browser SpeechRecognition API)
- Chat interface for agent interaction
- Modern dashboard UI inspired by Metronic
- Ready for future agent state/logs/controls

## Getting Started

1. Install dependencies:
   ```powershell
   npm install
   ```
2. Run the development server:
   ```powershell
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Voice-to-Text
- Uses the browser's built-in SpeechRecognition API (no AI required)
- Works in Chrome and most modern browsers

## Customization
- UI is based on Tailwind CSS and can be themed to match Metronic style
- Extend with agent state, logs, and controls as needed
